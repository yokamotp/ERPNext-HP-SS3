# よくある質問

## 🚀 導入に関する質問

<FAQ question="ERPNext は中小企業でも使えますか？">
はい、ERPNext は中小企業向けに設計されており、柔軟なカスタマイズが可能です。
初期費用も抑えられ、段階的な導入ができます。

**主なメリット:**
- ライセンス費用が不要
- 必要最小限の機能から開始可能
- 企業の成長に合わせて拡張可能
</FAQ>

<FAQ question="導入にかかる期間はどのくらいですか？">
導入の規模と複雑さによって異なりますが、基本的な機能であれば1-3ヶ月程度で運用開始が可能です。

**目安:**
- 小規模導入（基本機能のみ）: 1-2ヶ月
- 中規模導入（主要機能）: 2-4ヶ月
- 大規模導入（全機能・カスタマイズ）: 4-8ヶ月
</FAQ>

<FAQ question="データ移行は必要ですか？">
既存のシステムがある場合は、データ移行が必要になります。ただし、ERPNext は標準的なデータ形式に対応しているため、比較的スムーズに移行できます。

**対応形式:**
- CSV, Excel ファイル
- データベース（MySQL, PostgreSQL等）
- API 連携
</FAQ>

## 💰 コストに関する質問

<FAQ question="導入・運用コストはどのくらいですか？">
ERPNext はオープンソースのため、ソフトウェアライセンス費用は不要です。

**主なコスト:**
- サーバー・インフラ費用
- 導入・設定費用
- 運用・保守費用
- カスタマイズ費用（必要に応じて）

**目安:**
- クラウド版: 月額5,000円〜
- オンプレミス版: 初期100万円〜
</FAQ>

<FAQ question="無料版と有料版の違いは？">
ERPNext のソフトウェア自体は無料ですが、クラウド版では機能制限があります。

**無料版の制限:**
- ユーザー数制限
- ストレージ容量制限
- サポートレベル
- 高度な機能の制限
</FAQ>

## 🔧 技術に関する質問

<FAQ question="日本語対応はどうなっていますか？">
ERPNext は多言語対応しており、日本語のローカライゼーションも充実しています。

**日本語対応機能:**
- ユーザーインターフェース
- レポート・帳票
- メール通知
- ヘルプ・ドキュメント
</FAQ>

<FAQ question="クラウド版とオンプレミス版の違いは？">
それぞれにメリット・デメリットがあります。

**クラウド版:**
- メリット: 初期投資が少ない、メンテナンス不要、スケーラビリティ
- デメリット: カスタマイズ性が限定的、データ制御の制限

**オンプレミス版:**
- メリット: カスタマイズ性が高い、データの完全制御、セキュリティ
- デメリット: 初期投資が大きい、運用・保守が必要
</FAQ>

<FAQ question="API 連携は可能ですか？">
はい、ERPNext は RESTful API を提供しており、外部システムとの連携が可能です。

**API 機能:**
- データの取得・更新・削除
- ワークフローの実行
- レポートの生成
- リアルタイム同期
</FAQ>

## 👥 ユーザー・権限に関する質問

<FAQ question="ユーザー数に制限はありますか？">
クラウド版ではプランによって制限がありますが、オンプレミス版では制限がありません。

**クラウド版の制限例:**
- 無料版: 1ユーザー
- スタンダード版: 5ユーザー
- プロフェッショナル版: 無制限
</FAQ>

<FAQ question="権限管理は細かく設定できますか？">
はい、ERPNext は非常に細かい権限管理が可能です。

**権限設定項目:**
- モジュール・機能へのアクセス
- データの閲覧・編集・削除
- 承認権限
- レポート・ダッシュボードの表示
- フィールドレベルの権限
</FAQ>

**対応機能:**

</FAQ>
